# Mesh Messanger (React Native)

Аналог Meshtastic-клиента с UX в стиле привычного мессенджера:

- BLE подключение к LoRa контроллеру через `react-native-ble-plx`.
- Сквозная асимметричная шифрация `tweetnacl` (расшифровать может только получатель с приватным ключом).
- Онбординг: сначала BLE-контроллер, потом ввод ника.
- Друзья через QR-код (публичный ключ + id + ник).
- Локальное хранение профиля, друзей и переписок через `AsyncStorage`.
- Светлая/тёмная тема.
- Русский язык по умолчанию + переключение на English.

## Быстрый старт

```bash
npm install
npm run start
npm run android
# или
npm run ios
```

## Криптография

- При первом запуске создаётся пара ключей Curve25519 (`nacl.box.keyPair()`).
- Отправка: сообщение шифруется публичным ключом адресата + приватным ключом отправителя.
- Получение: расшифровка возможна только приватным ключом адресата и публичным ключом отправителя.

> В текущем прототипе транспорт по mesh-сети показан концептуально; добавьте реальную отправку ciphertext через BLE/Lora payload контроллера.
